<h1 align="center">Лабораторная работа № 2 <br> "Описание Лабораторной работы №1"</h1>
<h3>Условие задания:</h3>
<p>Создать файл sh и bat, который выполняет следующее: 
На вход пакетному файлу приходит относительный путь к папке (как параметр пакетного файла). 
Если такой папки нет, то писать “Данной папки нет” и завершить выполнение программы. 
В указанной папке должны содержаться различные файлы. Найти файлы, созданные определенным пользователем в определенный период 
(также задается как параметры пакетного файла), имена найденных файлов вывести в файл result.txt. 
</p>

<h3>Аргументы:</h3>
<ol>
<li>Имя папки(т.к. поиск производится относительно текущего каталога)</li> 
<li>Имя пользователя(в Bat-файле с доменом(имя ПК))</li> 
<li>Дата создания файла</li> 
</ol>
